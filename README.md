# Python Task Manager

A simple, command-line based task management application built in Python. This project allows users to track their daily tasks, view them, and save them to a file.

## Features
- **Task Management:** Add tasks and view your current list.
- **Data Persistence:** Tasks are saved to a `tasks.json` file so your data remains when you close the program.
- **Productivity Boost:** Displays a random productivity tip every time you launch the app.
- **Clean Code:** Uses modular functions and meaningful comments.

## How to Run
1. Ensure you have Python installed on your computer.
2. Clone this repository or download the `main.py` file.
3. Open your terminal or command prompt.
4. Run the program using: `python main.py`
